For a Command, use one back tick at the beginning and end of the command:
`ls -a`

For a Block of Code, use three back ticks at the beginning and end of the code:
```
<!DOCTYPE html>
<html>
<head> 
<title></title>
</head>
<body>
<body>
</html>
```
